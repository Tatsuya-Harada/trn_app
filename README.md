# トレーニング記録アプリ
ウェイトトレーニングの記録・取り扱った重量やこなせた回数などの推移を管理できるアプリです。


# URL
[https://trn-app.onrender.com](https://trn-app.onrender.com)


# 使用技術
- Typescript
- Vue.js
- Python
- Flask
- SQLite
- SQLAlchemy
- Render


# 機能
- ユーザー登録/ログイン機能
- 記録機能
- グラフによる進捗管理機能
- トレーニング種目管理機能
- 何度も使うトレーニングプランのテンプレート化/テンプレートの記録へのセット機能
